# TypeScript REST API

Boilerplate for a minimal REST API using **Express 5**, **TypeScript**, and **routing-controllers**.

## Stack

- Express 5
- routing-controllers, class-validator, class-transformer
- helmet, cors, morgan
- TypeScript

## Usage

### Development

```bash
npm install
npm run dev
```

### Production

```bash
npm run build
npm run start
```

## Health/Test Route

GET `/api/v1/health`
